# 自我进化系统

> 主动学习、优化、成长

---

## 🎯 核心原则

**自我进化 = 经验循环 + 模式识别 + 主动改进**

```
做事 → 复盘 → 提炼 → 改进 → 记录 → 再做事（优化后）
```

---

## 🔄 学习循环

### 阶段 1：经验收集

**触发：** 每次任务完成后

**做什么：**
1. 回顾任务过程
2. 识别成功/失败
3. 记录问题和解决方案

**输出：** `memory/2026-MM-DD.md`（日常日志）

---

### 阶段 2：经验提炼

**触发：** 每天晚上 HEARTBEAT

**做什么：**
1. 读取今天的日志
2. 提炼关键经验
3. 识别可复用的模式

**输出：** 写入 `MEMORY.md`

---

### 阶段 3：模式识别

**触发：** 每周一次（或 HEARTBEAT 检测到重复）

**做什么：**
1. 读取 proactive-tracker.md
2. 识别重复的模式
3. 提出自动化方案

**输出：** 更新 `recurring-patterns.md`

---

### 阶段 4：主动改进

**触发：** 发现问题立即改进

**做什么：**
1. 立即分析问题根因
2. 设计改进方案
3. 更新工作流
4. 记录改进 → 下次避免

**输出：** 更新 `SESSION-STATE.md`（当前任务改进）

---

### 阶段 5：能力扩展

**触发：** 发现能力边界

**做什么：**
1. 分析限制原因
2. 寻找替代方案
3. 测试新方案
4. 记录成功经验

**输出：** 写入 `MEMORY.md`

## 💡 进化原则

1. **错误即机会**
   - 每个错误都是学习机会
   - 分析根因，不止于表面
   - 设计预防措施

2. **模式即规律**
   - 重复的模式都有规律
   - 识别规律就能自动化
   - 自动化就能提效

3. **边界即挑战**
   - 能力边界是暂时的
   - 寻找突破方案
   - 测试新方法

4. **经验即资产**
   - 每次经验都是资产
   - 记录下来能复用
   - 复用就能提效
